GLEAMI 2.0 (기존)

구매 금액별 차등 배송비

  • 50만원 이하 : 4,000원
  • 50만원 초과~100만원 : 7,000원
  • 100만원 초과~199만원 : 10,000원
  • 199만원 초과~300만원 : 13,000원
  • 공동구매 합배송 불가
  • 주문 건당 배송비 중복 과금
GLEAMI 3.0 (변경)

왜 변경했나요?

결제 페이지 개발 중 주문 건당 배송비 중복 과금 + 주문서 개별 발행 이슈가 발생했습니다. 카페24 시스템 한계로 합배송 설정이 불가하여, 조건부 무료배송으로 전환하고 배송비를 판매가에 내재화하는 방향으로 결정했습니다.

🛒
카페24 묶음배송 분리 발송 가이드

공동구매 상품과 주얼리를 함께 구매한 복합 주문이 들어왔을 때,
카페24 배송 준비중 관리에서 공동구매 상품 / 주얼리(24K/투자용 포함) 상품을 분리하여 각각 송장을 할당하는 방법입니다.

1

복합 주문 확인

배송 준비중 관리 페이지에서 공동구매와 주얼리가 함께 포함된 주문을 확인합니다.
아래 예시처럼 하나의 주문번호에 공동구매 상품과 주얼리 상품이 함께 들어와 있습니다.
상품명을 확인하세요. 공동구매 상품과 주얼리/24K/투자용 상품이 한 주문에 섞여 있으면 분리 발송 대상입니다.
2

공동구매 주문 선택 → 묶음선택 → 송장번호 발급

같은 주문 내에서 공동구매 상품만 체크박스로 선택한 뒤,
묶음선택 체크 후 "송장번호저장" 버튼을 클릭하여 송장번호를 발급받습니다.
반드시 공동구매 상품끼리만 선택하세요. 주얼리/24K/투자용 상품과 섞어서 선택하면 안 됩니다.
3

공동구매 송장번호 할당 확인

공동구매 상품에 송장번호가 정상적으로 할당된 것을 확인합니다.
운송장정보 컬럼에 CJ대한통운 송장번호가 표시됩니다.
4

주얼리 주문 선택 → 송장번호 발급

이번에는 주얼리 상품만 체크박스로 선택한 뒤,
동일하게 "송장번호 다시 발급" 버튼을 클릭하여 별도 송장번호를 발급받습니다.
공동구매에 이미 송장이 할당되어 있으므로, 주얼리는 "송장번호 다시 발급"으로 진행합니다.
5

주얼리 송장번호 할당 확인

주얼리 상품에도 별도의 송장번호가 정상적으로 할당된 것을 확인합니다.
이제 하나의 주문에 2개의 서로 다른 송장번호가 할당된 상태입니다.
6

분리 발송 완료 — 배송 대기 관리에서 최종 확인

배송 대기 관리 페이지로 이동하면, 1개의 주문이 2개의 배송건으로 분리되어 있는 것을 확인할 수 있습니다.
각각 독립적인 송장번호로 발송 처리합니다.
결과: 고객에게는 공동구매 택배 1개 + 주얼리 택배 1개, 총 2개의 택배가 각각 배송됩니다.
배송비는 무료배송(10만원 이상)이므로 고객에게 추가 비용이 발생하지 않습니다.

요약 체크리스트

✅ 카페24 상품 배송비 세팅 → "조건부 무료" (10만원 이상 무료 / 미만 4,000원)
✅ 복합 주문(공동구매+주얼리) 발생 시 → 상품 유형별 분리하여 각각 송장 발급
✅ 고객 안내 메시지 → "10만원 이상 구매 시 무료배송"
✅ 배송 대기 관리에서 분리 발송 확인 후 출고
